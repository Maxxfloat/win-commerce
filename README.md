# Win Commerce

Hi this is a responsive e-commerce web app project

## Packages

- Nextjs
- Typescript
- Tailwindcss
- React-Query
- Next-Auth
- Axios

## So

It's a simple e-commerce web app with a Strapi backend for resumes.
with features as:

1.  Search bar
2.  Sidebar
3.  Profile

so I had not put too much time into it
